Doze ou Dez
==========================

A app to track game time and score when you are playing basketball with your friends, enjoy!

## Dependences

* nodejs
* ionic
* ionic

## Install

```bash
$ npm install
```

### Run

### Browse

```bash
$ ionic serve
```

### Android device

```bash
$ ionic run android
```
## Run Test

```bash
$ karma start
```
